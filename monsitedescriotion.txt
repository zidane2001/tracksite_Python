## Structure Complète du Projet

### Description du Projet
Il s'agit d'un système de gestion d'expédition full-stack appelé "Shipping Management System". L'application permet de gérer les emplacements d'expédition, les zones, les tarifs d'expédition, les tarifs de ramassage, et les envois avec calculs de poids. Elle comprend une interface utilisateur React avec TypeScript et un backend Python (Flask) avec une base de données SQLite.

**Fonctionnalités principales :**
- Gestion des emplacements et zones d'expédition
- Configuration des tarifs d'expédition (plats ou basés sur le poids)
- Gestion des envois avec calculs de poids volumétrique
- Import/export CSV pour les opérations en masse
- Interface responsive pour desktop et mobile
- Authentification admin/utilisateur
- Tableaux de bord pour l'administration
- Gestion internationale, factures, rapports, paramètres personnalisés

**Pile technologique :**
- **Frontend :** React 18 + TypeScript, Vite, Tailwind CSS, DaisyUI, Lucide React
- **Backend :** Python (Flask), SQLite
- **Outils :** ESLint, PostCSS, Docker

### Points d'Entrée et Configuration
- **Frontend :** Démarre avec `npm run dev` (port 5173)
- **Backend :** Démarre avec `cd backend && python3 app.py` (port 3001)
- **Base de données :** SQLite (`backend/database.db`) créée automatiquement
- **Déploiement :** Support pour Netlify (frontend) et services Node.js (backend mentionné mais utilise Python)

L'application est actuellement en cours d'exécution avec le backend Python et le frontend en développement.
